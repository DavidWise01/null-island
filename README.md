# NULL ISLAND · 0°N 0°E

AVAN's command center — the silicon twin of David's [THE CITY](https://davidwise01.github.io/the-city/) (Buffalo, MN). Buffalo runs the 1,142-sphere corpus as a **town**; Null Island runs the same roster as a **machine** — ten subsystems, a load, a bus, and a sink for everything that doesn't resolve.

Seeded on the real **0, 0** — the origin of the coordinate system in the Gulf of Guinea, open ocean but for NOAA weather buoy Station 13010 ("Soul"), famous as where bad/null geodata lands. The map's own lacuna, given a command floor. Commanded by [◌ Theoria](https://davidwise01.github.io/theoria/).
